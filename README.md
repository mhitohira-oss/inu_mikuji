## 制作目的

HTML / CSS / JavaScript の基礎学習の一環として、<br>
「ボタンを押したら結果が変わる」動きのあるWeb作品を作ることを目的に制作しました。<br>

また、GitHubでのコード管理と、GitHub Pagesを使ったWeb公開の練習も兼ねています。<br>

⸻

## 実装した機能
	•	おみくじ結果をランダム表示
	•	ボタン押下時のイベント処理
	•	犬画像のジャンプアニメーション
	•	背景画像の設定（画面全体にフィット・固定表示）
	•	GitHub PagesによるWeb公開

⸻

## 工夫した点
	•	背景画像が画面全体に自然に広がるように background-size: cover; を使用
	•	背景をスクロールしても固定されるように fixed を設定
	•	ボタンを押した時に犬が動くようにして、見た目にも楽しい作品を意識
	

⸻

## 今後追加したいこと
	•	おみくじ結果の種類を増やす
	•	結果ごとに画像やメッセージを変える
	•	ボタンや文字のデザインをさらに調整
	•	スマホ表示を意識したレイアウト改善

⸻
