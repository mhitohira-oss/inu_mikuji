## いぬみくじ
JavaScriptの学習用に制作した、シンプルなWebアプリです。<br>
ボタンを押すとランダムでおみくじ結果が表示され、犬の画像がぴょんぴょん跳ねるアニメーションを入れています。<br>
 
## 公開URL
https://mhitohira-oss.github.io/inu_mikuji/

- HTML
- CSS
- JavaScript
- GitHub
- GitHub Pages
 
## 実装した機能
おみくじ結果をランダム表示<br>
ボタン押下時のイベント処理、画像のジャンプアニメーション<br>
背景画像の設定<br>
 
## 工夫した点

また、GitHubでのコード管理と、GitHub Pagesを使ったWeb公開の練習も兼ねています。<br>
